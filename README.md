# xrshop_h5

#### 项目简介
xrshop是基于当前流行技术采用前后端分离商城系统,包含商城,会员,充值,购物车,退款,订单,付款等功能，更适合企业,个人二次开发；


#### 体验地址

|   官网地址  |    http://shengxian.judaixia.com 
|---  |--- | --- |


## 功能

* 一：商品模块：商品添加、规格设置，商品上下架等
* 二：订单模块：下单、购物车、支付，发货、收货、评价、退款等
* 三：配置模块：各种配置
* 四：用户模块：登陆、注册、会员卡、充值、购买等
* 五：其他等
